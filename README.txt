COWS AND BULLS GAME:

Welcome to the "Cows and Bulls" game! Can you guess the secret 3-digit number?

HOW TO PLAY?!

.The system will randomly generate a 3-digit number with distinct digits.
.Your task is to guess this number within 5 attempts.
.Each guess should be a 3-digit number with distinct digits.

CLUES:

Bull: If a digit in your guess matches both in value and position with the system-generated number, it's a bull.

Cow: If a digit in your guess matches the value but is in a different position than the system-generated number, it's a cow.

WINNING:

You win the game if you manage to guess the number with exactly 3 bulls and 0 cows within the 5 attempts.
